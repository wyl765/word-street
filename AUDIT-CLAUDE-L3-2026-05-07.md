## 1. Definition Errors & Inaccuracies

### L3a
| Word | Issue |
|------|-------|
| `careless` | Definition says "not paying **focus**" ŌĆö should be "not paying **attention**". "Focus" is not idiomatic here. |
| `native` | Definition: "**at first** from a certain place" ŌĆö should be "**originally** from a certain place". |
| `radical` | Definition: "very **unlike** from what's normal" ŌĆö should be "very **different** from what's normal". |
| `reckless` | Definition: "with no **caring** about safety" ŌĆö should be "with no **care** about safety" or "without caring about safety". |
| `influential` | Definition truncated: "the power to change opinions or **do**" ŌĆö missing object. Should be "opinions or actions" or similar. |
| `microscopic` | Definition: "you need a special tool to see it" then example says "too small to see without a **small lens**" ŌĆö should be "**microscope**" not "small lens". Confusing for a 10-year-old. |
| `citadel` | Example: "The ancient citadel sat **covering** the hill" ŌĆö should be "**crowning**" or "atop". |
| `imaginary` | Example missing comma: "her imaginary friend**,** a purple dragon" |
| `caldron` | Nonstandard spelling. Standard modern English is "**cauldron**". At minimum note the variant. |

### L3b
