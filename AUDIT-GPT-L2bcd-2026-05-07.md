### A) Level / age-appropriateness mismatches (too advanced for target)
These entries are far beyond typical 10-year-old ESL (MAP 197) in concept load, morphology, or academic register.

- **moreover / nevertheless / nonetheless / accordingly / consequently / namely / notably / on the contrary / regardless**: discourse markers are advanced; examples require high reading maturity.
- **adolescent**: concept/wordform likely above level; overlaps with “teenager”.
- **cellulose, chromosome**: biology terms typically far above level; definitions include technical framing (“cell walls”, “cells that carries body instructions”).
- **acoustics, alloy, amass, anthem, aperture, aroma, assent, attest, balk, bang out, bedrock, bestow, blot out, bog, botany, bout, brackish, bristle, brunt, bumble, capillary, chasm, clump, compost, concave, convex, crevice, decoy, detour, dilute, dwindle, eddy, engulf, extricate, gust, morph, scour, silt, spore, submerge, unearth, verdant, sapling, topsoil**: overall difficulty and abstraction too high for target.

### B) Definition accuracy / completeness problems
- **myth** — Definition “explains how something in nature works” is too narrow; myths often explain origins, beliefs, gods, heroes (not only nature).
- **vowel** — Definition “the letters a, e, i, o, u” is incomplete; **Y** is sometimes a vowel in English (e.g., “my,” “gym”).
- **carbon** — Defined as “a material found in all living things.” Carbon is a **chemical element**; it is found in many non-living substances too. Example mixes “air we breathe out” (CO2) with “coal” without clarifying.
- **virus** — “a very tiny thing that can make you sick” is vague/childish and scientifically sloppy; should be “a tiny germ that causes disease” (and is not a living thing like bacteria).
- **latitude / longitude** — Definitions “lines on a map that go side to side / up and down” are too vague and can mislead; should reference imaginary lines on Earth/globe and north–south/east–west position.
- **water cycle** — Example “includes drying up, clouds, and rain” is inaccurate/oversimplified; missing key stages (evaporation/condensation/precipitation/collection). “Drying up” is not the standard term.
- **gravitational** — Example uses “Gravitational pull…” but target word is “gravitational” (adjective). The example’s main subject is “pull” and may confuse POS usage.
- **acoustics** — Definition “how sound moves in a room” is oversimplified; acoustics is the science of sound (incl. reflection, absorption). Also too advanced.
- **aperture** — “an opening or hole, mainly in a camera” is incomplete; for camera context should mention controlling light.
- **bout** — Definition “often illness or action” is unclear; typically “a short period of illness or intense activity” (needs clearer framing).
- **brunt** — Example uses hurricane; definition “main force” is OK, but abstractness too high for level.
- **extricate** — “tangled case” should be “tangled situation” (wrong collocation).

### C) Example quality / grammar / usage problems
- **conflict** — Example is awkward: “could not find home” → should be “could not find **its way** home” / “could not find **his** home.”
- **consistently across multiple entries (connectors list)** — Many examples use semicolon-heavy academic style (e.g., “It was very hot; nevertheless…”). This register is likely too hard for the target reader.
- **use up** — Definition grammar error: “to finish all something” → “to use all of something / to finish all of something.”
- **work out** — Definition lists “exercise, or solve a problem,” but example uses a different sense (“Everything worked out fine” = ended well). Definition/example mismatch.
- **make out** — Definition “to see or get something with trouble” is unidiomatic; should be “with difficulty.”
- **take into account** — Example “Take the cost into account before buying.” is incomplete (buying **what**?).

### D) imageKeyword clarity / mismatch problems
- **regroup** — imageKeyword “kids trading lunch snacks” is unrelated to math regrouping; not a reliable retrieval keyword.
- **drop off** — imageKeyword “mailman delivering package door” mismatches the example (dropping off lunch / leaving someone/something).
- **look up** — imageKeyword “flashlight searching dark room” mismatches dictionary lookup meaning.
- **especially** — imageKeyword is a full sentence-length scene (“child pointing excitedly…”); not a clean search keyword.
- **at the same time** — imageKeyword “friends holding hands circle” doesn’t depict simultaneity; likely off-target.

### A) Definition accuracy / wording problems
- **distinct** — Definition “clearly unlike from others” is ungrammatical and unclear. Should be “clearly different from others.”
- **duty** — Definition has duplication/error: “because it is your job or job.”
- **liable** — Definition “in charge of something or likely to happen” incorrectly merges two different senses. “Liable” = legally responsible (or “likely” in a different construction). Example is the legal-responsibility sense only.
- **fault** — Definition is wrong: “job for something that went wrong.” Should be “responsibility/blame for something wrong.”
- **forecast** — Definition is weather-only (“future weather”), but example is **financial**. Either broaden definition to “prediction about the future” or change example to weather.
- **function** — Definition is general “job/purpose,” but example is coding-specific; imageKeyword also mismatched (see below). Consider either a general example or a clearer split of senses.
- **immediate** — Definition ungrammatical: “with no any delay” → “with no delay.”
- **imply** — Definition is broken/incomplete: “without saying it right at” (missing “right away/out loud”).
- **impress** — Definition inaccurate: “make someone think you are perfect.” “Impress” = make someone admire/respect you.
- **incident** — Definition awkward and inaccurate: “most times something odd.” Incidents can be any event, often especially unusual/important.
- **institution** — Definition “an important group like a school or hospital” is too loose; institutions are organizations/establishments (or long-standing systems).
- **license** — Definition incorrect/unnatural: “official okay to act.” Should be “official permission.”
- **maximum** — Definition ungrammatical: “the most or largest amount can be done” (missing words; also “amount” vs “number”).
- **novel** — Definition ungrammatical: “unlike from anything before” → “unlike anything before.”

### B) Example grammar / punctuation problems
- **concern** — Example grammar error: “made his check” → “made him check.”
- **consist** — Example punctuation/listing error: “lettuce tomatoes, and carrots” → “lettuce, tomatoes, and carrots.”
- **champion** — Example missing punctuation: “After winning every race she became…” → comma after “race”.
- **dilemma** — Example has unnecessary comma: “between the field trip, and the party.”
- **dimension** — Example uses plural “dimensions” while the target word is singular “dimension” (form mismatch for learners).
- **fault** — Example OK for meaning, but conflicts with wrong definition.
- **issue** — Example ungrammatical/unclear: “Pollution or water is a big issue that affects all.”
- **institution** — Example ungrammatical: “helps all learn.”

### C) imageKeyword clarity / mismatch problems
- **function** — imageKeyword “bridge function” is unrelated/unclear (and “function” isn’t a bridge). Misleading retrieval.

### A) Definition accuracy / clarity / register problems
- **peril** — Definition “something that can hurt or kill you” is too narrow/over-dramatic; peril = serious danger/risk.
- **phenomenon** — Definition “something odd or cool that happens” is informal and inaccurate; a phenomenon is an observable event/fact (not necessarily odd/cool).
- **plea** — Definition ungrammatical: “a serious and full of feeling request.” Should be “a serious, emotional request.”
- **reflect** — Definition has a typo/format error: “bounce backlight” → “bounce back **light**.” Also mixes two senses; example only shows mirror sense.
- **prompt** — Example uses “prompt reply,” but many learners confuse adjective vs verb “prompt.” Needs clearer part-of-speech framing (currently only adjective).
- **prospect** — Example grammar error: “The prospect of **summer trip**…” → “the prospect of **a** summer trip…”
- **standard** — Definition ungrammatical: “the normal level of how good expected.”
- **threaten** — Definition is person-only (“say you will hurt…”), but example uses weather (“clouds threatened…”). Definition must include “seem likely to cause harm.”
- **trend** — Definition wording: “well liked style” → “well-liked style”; also should emphasize “a general direction of change” (not only style).
- **valid** — Definition “right and true” is too broad; “valid” can mean “legally acceptable / still usable / logically sound.” Example is ticket validity; definition should match.
- **vital** — Definition “fully needed” is unidiomatic; should be “absolutely necessary.”
- **warrant** — Definition OK, but example is awkward: “enough to warrant a closer look by the teacher” is clunky for the target.

- **certify** — Example is semantically odd: doctors typically don’t certify that food is safe to eat (health inspectors/labs do). Misleading context.
- **cumulative** — Definition typo: “overtime” → “over time.”
