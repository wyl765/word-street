## 4. 让我觉得"对儿子不合适"的内容

1. **"drown" (Level 2)** — "to go under water and not be able to breathe" 这个定义对10岁有点吓人。虽然例句讲了life jacket是安全教育，但放在词汇游戏里让我不太舒服。

2. **"attack" (Level 2)** — 虽然例句是 goose attacking sandwich（很可爱），但定义 "to try to hurt" 比较暴力。对于游戏还行，但我注意到了。

3. **"destroy" (Level 2)** — "to break so it cannot be used" + 沙堡例子，OK可以接受。

4. **"dagger" (Level 2)** — "a short pointed knife" + 海盗例子。作为妈妈…我不太喜欢武器类词汇出现在学习游戏里，但我知道这是幻想题材，勉强接受。

5. **"trap" (Level 1)** — "something used to catch animals" 对动物爱好者的孩子可能不舒服，但教育层面OK。

**总体：** 没有真正不合适的内容，只是 drown 让我稍有顾虑。

### 高风险混淆组：

| 组 | 词 | 原因 |
|---|---|---|

| **动作词** | leap / skip / hop / jump / dash | 五个都是"快速移动"，细微差别2年级把握不了 |
| **声音词** | shout / roar / howl | 都是"大声叫"，区别对2年级偏难 |
| **湿度词** | damp / soaking / wet | damp vs soaking OK（有程度差），但和例句/图重叠 |

| **大小词** | huge / enormous / gigantic | 三个都是"非常大"，2年级不需要区分 |
| **情绪词** | nervous / anxious / worried | 三个都是"担心"，anxious在Level 2还好，但和Level 1的nervous/worried叠了 |

## 6. 总结 & 优先建议

### 必须改的（High Priority）：
1. **把 although, whether, unless, seldom, eventually, throughout 从 Level 1 移到 Level 2** — 这些是抽象连接词/副词，2年级不该碰
2. **修 unless 的定义** — "only if not" 没有任何2年级能读懂
3. **修 fjord 的定义** — inlet 和 cliffs 超纲
4. **给 seldom, perhaps, damp, slimy 换更具象的 imageKeyword** — 目前搜不出有意义的图
5. **解决 beneath/below/under 三词重叠** — 删掉一个或合并

### 建议改的（Medium Priority）：
6. 把 whether/weather, heal/heel, tale/tail 标记为"同音词对"并在游戏中同时教学，而不是随机出现
7. 给所有动作词(leap/skip/hop/dash等)配更有区分度的 imageKeyword
8. moss/lichen 视觉太像，考虑删除 lichen 或换成别的词
9. huge/enormous/gigantic 三个留两个就够

### 可以接受的：
- 大部分定义和例句对2年级是合适的
- 例句长度整体控制得好
- Level 1 的动物和食物类词汇非常棒，Mark会喜欢
- imageKeyword 大部分设计得很巧妙（比如 "toy in a box with name tag" for belong）
