## words-level4a.js (LEVEL4A_BANK)

### Definition accuracy / clarity problems
- **atom** — Definition is scientifically inaccurate: atoms are **not** “the tiniest particle that makes up all matter.” Better: “the smallest unit of an element that still has that element’s properties.”
- **civic** — Definition is too narrow (“duties citizens have in their town”). “Civic” relates broadly to citizens/citizenship and community/public life; not just duties.
- **client** — Definition is truncated/incomplete: “a person who pays for a service from a skilled” (missing the noun, e.g., “professional”).
- **excel** — Definition is wrong: “to be perfect at something.” “Excel” means **to do very well / be very good**, not “perfect.”
- **dissidence** — Definition is incorrect/unnatural: “fight with official rules or opinions.” Should be closer to “disagreement/opposition to official policy (often political).”
- **dogmatic** — Definition is unclear and ungrammatical (“not looking at others”). Should be “stubbornly believing you are right and refusing to consider other views.”
- **elucidation** — Definition is wrong: it is not “a reason”; it is **the act/process of explaining/making clear**.
- **enormity** — Definition is risky under strict publisher standards: common primary meaning is **great wickedness/serious wrongdoing**; using it as “great size” is widely criticized/treated as secondary.
- **eradication** — Definition is off (“complete ruin or removal”). “Eradication” is **complete removal/destruction (so it no longer exists)**; “ruin” is not the core meaning.
- **expediency** — Definition is inaccurate (“hands on”). Expediency is “convenience/practical advantage (sometimes over principle),” not “hands-on.”
- **explication** — Definition is incorrect/ungrammatical: “a detailed reason of something.” Should be “a detailed explanation.”
- **extenuating** — Definition is wrong: “making a mistake or offense seem less serious” is not the meaning. “Extenuating” refers to **circumstances that lessen blame/seriousness**.
- **adjunct** — Example misuses the word: “built an adjunct room” is not idiomatic. “Adjunct” is typically “an adjunct to X” (something added/attached), or “adjunct professor,” etc.

### Example sentence grammar / usage problems
- **assess** — Example is missing punctuation/structure: “assess the injured bird deciding…” → needs a comma or rewrite.
- **conveyance** — Example has subject–verb agreement error: “The horse and carriage **was**…” → should be “were,” or “A horse and carriage was…”.
- **eclipse** — Example has incorrect comma placement: “between Earth, and the sun” (no comma).
- **federal** — Example is incomplete/ungrammatical: “apply to all in the entire country” (missing noun: “all **people/citizens**…”).
- **global** — Missing comma: “affects every country, not just…”
- **illuminate** — Missing comma: “illuminate the cave, making…”
- **inaugurate** — Example uses wrong frequency wording: “each four years” → “every four years.”
- **inevitable** — Example is awkward and grammatically off (“After eating three slices of cake a stomachache felt inevitable.”). Needs comma + more natural phrasing.
- **erstwhile** — Example is ungrammatical: “returned … a year away” (missing “after”).
- **importune** — Example is ungrammatical: “kept on to importune …”
- **inexorable** — Example is ungrammatical/incorrect usage: “means all grows older” (“everyone grows older”).
- **incandescence** — Example phrasing is ungrammatical: “faces of all sitting around it” (should be “all **those** sitting…”).
- **lassitude** — Example is ungrammatical: “filled all with lassitude” (should be “filled everyone…”).
- **draconian** — Example incorrectly capitalizes “Draconian” mid-sentence; should be lowercase unless treated as a proper adjective in a specific style.

### imageKeyword problems (too abstract / non-visual)
- **comprehend** — “understanding concept” is abstract and hard to depict.
- **enhance** — “improving quality” is abstract.
- **feasible** — “possible plan” is abstract.
- **cognizant** — “aware person thinking” is vague/abstract.
- **deduction** — “detective thinking” is vague (could be made concrete: “detective with clue board”).
- **discretionary** — “choosing options” is abstract.
- **incisive** — “sharp thinking” is abstract.

## words-level4b.js (LEVEL4B_BANK)

### Definition accuracy / clarity problems
- **mimic** — Definition has a duplicated word: “to copy or copy…”
- **momentum** — Definition is scientifically incorrect: momentum is **not a force**; it is “quantity of motion” (mass × velocity).
- **retail** — Definition is ungrammatical: “the selling of goods right at to customers…”
- **sovereign** — Definition is broken (“having supreme power and power; free”). Duplicates “power” and is unclear.
- **substantial** — Definition is ungrammatical: “large, amount, or importance.”
- **transaction** — Definition is missing “and” and is ungrammatical: “goods services, or money…”
- **transition** — Definition has duplicated phrase: “from one state or state to another.”
- **sanction** — “official okay” is too informal/vague; also the two-sense word is confusing for this level without clearer framing.

### Example sentence grammar / usage problems
- **menace** — Example is incomplete: “biting all” (should be “biting everyone/people”).
- **monopoly** — Example is missing punctuation: “When one company has a monopoly, customers…”
- **obviate** — Example likely misuses the word: seatbelts reduce risk; “obviate serious injuries” implies making injuries unnecessary/impossible.
- **pragmatism** — Example pronoun error: “led **his** to study” → “led **him** to study.”
- **prudence** — Example pronoun error: “Prudence told **his**…” → “told **him**…”
- **pulchritude** — Example uses “all’s,” which is ungrammatical.
- **solace** — Example pronoun error: “gave **his** solace” → “gave **him** solace.”
- **substantial** — Definition is broken (see above).
- **terminal** — Example is missing “before”: “two hours **before** our flight…”
- **reverberate** — Definition is ungrammatical: “to be repeated as an echo have lasting effects” (missing “or/and”).
- **rescind** — Definition is missing punctuation/words: “cancel a law order, or choice” (needs commas/“an order”).
- **sagacious** — Definition appears truncated: “good judgment and getting” (unfinished).
- **warranted** — Example is missing a connector: “was warranted **because**…”
- **revelry** — Example missing commas: “included music, dancing, and fireworks.”
- **truculent** — Example is ungrammatical: “picked arguments with all” (should be “with everyone/anyone”).
- **vapid** — Example is ungrammatical: “so boring that all changed the channel” (should be “everyone”).

## words-level4c.js (LEVEL4C_BANK)

### Definition accuracy / clarity problems
- **correlation** — Definition is misleading (“connection or bond”). Under strict standards it should indicate a **relationship in which two things vary together**; “bond” suggests causation.
- **fraud** — Definition uses informal/unclear wording (“money or edge”). “Edge” is slang; use “advantage.”
- **insignificant** — Definition contains a critical typo/bug: “unkey” (should be “unimportant”).
- **memoir** — Definition is incorrect/unnatural: “memories and feels” (should be “memories and feelings/experiences”).
- **consecutive** — Definition is missing “to”: “next **to** each other…”
- **requisite** — Definition is broken/repetitive: “needed or needed…”
- **stagnant** — Definition contains duplicated phrase: “not growing or growing.”

### Example sentence grammar / usage problems
- **archive** — Example missing article: “from **over a** century ago.”
- **confederation** — Example is ungrammatical/awkward: “to fight for being free together.”
- **disposition** — Example is incomplete: “wags its tail at all” (should be “at everyone/anyone”).
- **forfeit** — Definition is missing “and”: “as a penalty **and** give it up…”
- **overhaul** — Example is ungrammatical: “putting in place of worn pieces” (missing words like “replacing worn parts”).
- **procurement** — Example misuses wording: “procurement part” (should be “procurement department/team/office”).
- **protocol** — Example is ungrammatical and missing “before”: “Safety protocol **requires** all visitors to sign in at the front desk **before** entering…”
- **perennial** — Example is ungrammatical: “come back each spring and not needing…”
- **taxonomy** — Example is incorrect/garbled: “down to one person species” (should be “down to one species”).
- **prospectus** — Definition is ungrammatical: “a paper giving facts about a school firm, or investment” (missing punctuation/wording).
- **renaissance** — Definition is missing commas/structure (“cultural artistic rebirth”).
- **replenish** — Example is ungrammatical: “replenish … starting the second half” (should be “before starting the second half”).
- **reproach** — Definition is wrong word choice: “express letdown” (should be “disapproval/disappointment/criticism”).
